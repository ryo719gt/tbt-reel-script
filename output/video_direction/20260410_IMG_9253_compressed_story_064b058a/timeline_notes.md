# Premiere Pro 編集ガイド｜TBT ナレーション先置き構成

> **編集の基本方針**
> ナレーション音声をタイムライン上に先に並べ、ナレーションの呼吸・間に合わせて映像を当てていく。テロップはナレーション音声のタイミングに追従させる。BGMはナレーショントラックの下に敷き、ナレーションより必ず低い音量で維持する。

### ▶ cue_id: 1 ／ hook
**累積タイムコード：0:00.0 〜 0:02.4**

#### 🎙 ナレーション
```
また、棚に戻してしまった。
```
- 推定尺：**2.4秒**
- 読み方メモ：最初の「また、」で0.3秒ほど間を置く。淡々と、後悔でも諦めでもなく、ただそうなってしまった、という語感で読む。語尾は上げない。

#### 📝 テロップ
```
また、棚に戻してしまった。
```
**表示ユニット（フェードイン順）：**
1. `また、棚`
2. `戻してしまった。`

- フォント：細めのゴシック系（UD黎ミン / Hiragino Sans W3 など）
- 配置：画面下部 中央寄せ
- 登場タイミング：ナレーション開始と同時 or 0.2秒遅れ
- ユニット2は「棚に戻してしまった」のナレーションが発話されるタイミングで切り替え

#### 🎬 映像クリップ
📷 **要撮り直し**

| 項目 | 内容 |
|------|------|
| source_file | *(未指定)* |
| source_timecode | `--:--` |
| 推奨カット内容 | バッグを手に持ちかけて、そっと棚やクローゼットの棚板に戻す手元のクロースアップ。引き戻す動作が見えること。照明は落ち着いた室内光。ピントは手とバッグに。 |
| カット長 | 2.4秒 |
| カメラワーク | 固定 or 極めて微細なスライド |

#### 🎵 BGM
- このセクションからBGMをフェードイン（0:00.0〜0:02.0 で徐々に立ち上げる）
- 音量目安：ナレーションが −3dBとすれば BGM は −18〜−22dB 程度
- 推奨テイスト：ピアノまたはアコースティックギターの単音系。テンポ感のないアンビエント寄り

### ▶ cue_id: 4 ／ hesitation
**累積タイムコード：0:10.6 〜 0:18.2**

> ⚠️ evidence_confidence: unconfirmed

#### 🎙 ナレーション
```
まだ売るとは、決めていない。決めていないのに連絡していいのか、それが手を止めていた。
```
- 推定尺：**7.6秒**
- 読み方メモ：「まだ売るとは、決めていない。」は静かに、自分に言い聞かせるように。「決めていないのに連絡していいのか、」でわずかに声のトーンが落ちる。「それが手を止めていた。」は過去形として、少し距離を置いた口調で。

#### 📝 テロップ
```
決めていないのに、連絡していいのか。
```
**表示ユニット（フェードイン順）：**
1. `まだ売ると`
2. `決めていない。`
3. `決めていないの`
4. `連絡していいのか。`

- ユニット1・2：ナレーション前半「まだ売るとは、決めていない。」に合わせて順次登場
- ユニット3・4：ナレーション後半「決めていないのに連絡していいのか、」に合わせて切り替え（前のユニットはフェードアウトしてから次を出すか、画面を一度クリアしてから再表示）

### ▶ cue_id: 5 ／ concrete_moment
**累積タイムコード：0:18.2 〜 0:26.0**

> ⚠️ evidence_confidence: unconfirmed

#### 🎙 ナレーション
```
相場を聞くことは、売ると決めることじゃない。そう気づいたとき、少し手が動いた気がした。
```
- 推定尺：**7.8秒**
- 読み方メモ：「相場を聞くことは、売ると決めることじゃない。」は穏やかに、しかしはっきりと。断言ではなく「気づき」として届けること。「そう気づいたとき、」で0.4秒ほど間を置く。「少し手が動いた気がした。」は控えめに、そっと言う。

#### 📝 テロップ
```
相場を聞くことと、売ると決めることは違う。
```
**表示ユニット（フェードイン順）：**
1. `相場を聞くことと、`
2. `売ると決めることは違う。`

- ユニット1：ナレーション開始と同時 or 0.3秒遅れ
- ユニット2：ナレーションの「売ると決めることじゃない。」に合わせて登場（目安：開始から約3.5〜4.0秒後）
- テロップはこのセクションで最も長く画面に残す（読み取る時間を確保する）
